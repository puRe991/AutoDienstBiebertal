# Auto Dienst Biebertal Website

Individuelle, statisch generierte Website für die KFZ-Meisterwerkstatt Auto Dienst Biebertal.

## Entwicklung

```bash
npm install
npm run dev
```

## Qualitätssicherung

```bash
npm run build
npm audit --audit-level=high
```

## Wichtige Hinweise vor Livegang

- Impressum und Datenschutz müssen rechtlich geprüft und mit vollständigen Unternehmensdaten ergänzt werden.
- Öffnungszeiten, Geo-Koordinaten und Teamdaten sind vor Veröffentlichung gegen offizielle Angaben zu verifizieren.
- Echte Werkstatt- und Teamfotos sollten die abstrakte Hero-Grafik perspektivisch ersetzen.
